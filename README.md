### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/spliteasy.git
   cd spliteasy
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn
   ```

3. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
